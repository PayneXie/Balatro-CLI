## 🇺🇸 English

### Why this exists?
**To play Balatro while pretending to work.**

This is a minimal, text-based CLI implementation of the hit game [Balatro](https://www.playbalatro.com/). It runs entirely in your terminal, making it look like you are coding, debugging, or managing servers, while you are actually hunting for that sweet Royal Flush.

The game logic is heavily inspired by and referenced from the amazing [balatro-gba](https://github.com/GBALATRO/balatro-gba) project.

### Features
*   **Stealth Mode**: Pure text interface. No flashy graphics. Looks just like a regular shell session.
*   **Core Loop**: Blind selection -> Hand playing/discarding -> Scoring -> Shop (WIP).
*   **Poker Hands**: Full implementation of Balatro's hand evaluation (High Card to Royal Flush).
*   **Scoring**: Authentic chips and mult calculations.

### How to Play
1.  **Install Dependencies**:
    ```bash
    pip install rich
    ```
2.  **Run (IMPORTANT!)**:
    > **⚠️ Do NOT run inside your IDE's terminal.**
    >
    > Please open a standalone terminal window (PowerShell, CMD, Windows Terminal) to play. The TUI needs full control of the screen buffer, which IDE terminals often lack.

    ```bash
    python main.py
    ```

3.  **Controls**:
    *   `Arrow Keys`: Move cursor
    *   `Space`: Select/Deselect cards
    *   `Enter`: Play Hand / Confirm Action
    *   `Q`: Quit (Boss is coming!)
